## How to use

```bash
  # Clone this repository
  git clone https://github.com/wilsonmjunior/go-explorer-repositories
  
  # Access the project folder 
  cd go-finances-web
  
  # Install dependencies
  yarn 
  
  # Run the application
  yarn start
  
```
